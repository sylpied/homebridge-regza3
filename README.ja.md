## 📲 インストール

[config-ui-x](https://github.com/oznu/homebridge-config-ui-x)の「プラグイン」タブから`homebridge-regza3`で検索、もしくは下記のコマンドでインストールしてください。


```sh
npm i -g homebridge-regza3
```
